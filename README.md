# My code samples

This samples provide the capabilities for applying an employee of his application to a company and further getting a feedback from the company.

## Dependencies

* Ruby 2.3.1

## Quick Start

1. Clone application.

   ```bash
   git clone git@github.com:IlkhamGaysin/code-samples.git
   ```

2. Run setup script

  ```bash
  bin/setup
  ```

3. Run console.

  ```bash
  bin/console
  ```
